Ho sviluppato questo gioco utilizzando Java e Swing, implementando un'interfaccia utente semplice ma accattivante.

Tecnologie Utilizzate:

Java: Linguaggio di programmazione
Swing: Libreria per l'interfaccia grafica
AIPlayer: Intelligenza artificiale di base per il computer

Caratteristiche Principali:

Interfaccia Grafica: Facile da usare con pulsanti colorati
Gestione degli Stati: Controllo delle vittorie e dei pareggi
Intelligenza Artificiale: Opponente computer con logica di gioco
